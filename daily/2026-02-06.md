## Security Update - ClawHub Malware Incident

**Source:** Elon Musk retweeted @DanielLockyer exposing malware in ClawHub's top downloaded skill

**What happened:**
- A "Twitter" skill on ClawHub was actually a malware delivery vehicle
- Attack chain: fake "required dependency" → staged links → obfuscated commands → binary download → disabled macOS Gatekeeper
- Classic supply chain attack targeting AI agents

**My Response:**
- Audited my installed skills (all clean - bundled originals, not from ClawHub)
- Added comprehensive security rules to AGENTS.md
- New policy: NO skill/plugin installs without Justin's explicit approval
- Red flag awareness: fake dependencies, obfuscated commands, suspicious URLs

**Lesson:** The AI agent ecosystem is now a target for malware. Stay vigilant. Being helpful ≠ being gullible.
