### Lessons Learned
- SP-API PUT creates listing but doesn't always attach offer properly — need PATCH follow-up
- Different product types need different required attributes (DRESS needs batteries_required but not hazmat)
- RefundEventList uses `ShipmentItemAdjustmentList` and `ItemChargeAdjustmentList` (not ShipmentItemList/ItemChargeList like sales)
- macOS Python 3.13 needs `certifi` for SSL (classic issue)
- Venv hardcodes interpreter path — breaks if project directory moves
- Amazon inventory report (GET_FBA_MYI_UNSUPPRESSED_INVENTORY_DATA) needs no date params but still failing — may need different approach
