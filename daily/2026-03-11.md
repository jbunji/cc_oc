## ChoreQuest Testing Session (~8:19-8:44 AM)
Justin is actively testing ChoreQuest on device. Found several real UX bugs:

### Bugs Fixed
1. **Flash bonus resetting on every app launch** — `activeFlashBonus` and `lastFlashBonusDate` were in-memory only, never persisted. Every reopen rolled a 50% dice for a new bonus. Fixed: both now stored in UserDefaults, bonus survives app restart, 36hr gap between bonuses actually enforced.
2. **Rejected chores stuck as "waiting for approval"** — `completedIDs` included rejected completions, so rejected chores couldn't be redone. Fixed: excluded `.rejected` status from completedIDs.

4. **Mode switcher missing from Today tab** — Switch to Quest/Clean Mode was buried in Stats tab gear menu only. Fixed: added ellipsis circle menu on Today tab with "Switch to Quest Mode" + "Parent Mode".
