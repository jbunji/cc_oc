## Evening Session — API Key Swap & App Store Submission (~3:20 PM - 6:15 PM)

### API Key Swap (Painful but Done)
- Justin bought new Claude Max subscription on separate Google account to save money
- Swapping the API key in auth-profiles.json was a nightmare:
  - The gateway caches keys in memory and flushes them back to disk on graceful shutdown
  - `openclaw gateway restart` and `SIGUSR1` both do graceful shutdown → overwrites edits
  - The key existed in 8+ agent auth-profiles.json files (main, luna, bolt, sage, nova, auditor, migrator + backup copy)
  - `openclaw configure` wizard wrote to wrong directory (`/Users/justinbundrick/.openclaw/` instead of `/Users/openclaw/.openclaw/`)
  - Gateway runs as `justinbundrick` but reads config from `/Users/openclaw/.openclaw/`
  - **Solution that worked:** `kill -9` (no graceful shutdown) → edit file → `openclaw gateway start`
  - Justin spent ~$12 on API charges during the debugging — was frustrated
- New Anthropic token: starts with `ST2P8OHec5` (old one started with `rgLSloX4`)
- Documented the procedure for future reference
