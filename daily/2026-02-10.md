## Cron Investigation - ROOT CAUSE FOUND
- Cron jobs haven't worked since moving to Mac Mini
- **Root cause:** `runMissedJobs()` in `/src/cron/service/timer.ts` runs missed jobs **sequentially with await**
- If any job hangs (API timeout, network issue), entire cron service blocks because it holds a mutex lock
- All subsequent cron operations (list, status, run) timeout waiting for the lock
- This is a bug that should be reported — runMissedJobs needs timeouts or non-blocking execution
- Manual `--force` runs work; automatic timer firing does not
