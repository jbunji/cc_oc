### Architecture
```
┌─────────────────────────────────────────────────┐
│                  Pinch iOS App                  │
├─────────────────────────────────────────────────┤
│  🎤 iOS Speech → text (SpeechRecognizer.swift)  │
│  📡 WebSocket to ws://127.0.0.1:18789           │
│     └── Protocol: connect.challenge → connect   │
│     └── Then: chat.send → event:chat response   │
│  🔊 iOS TTS / OpenAI / ElevenLabs               │
└─────────────────────────────────────────────────┘
```

### Files
- `PinchApp.swift` - Entry point
- `Config.swift` - Gateway URL, TTS backend selection
- `ContentView.swift` - Main UI
- `SpeechRecognizer.swift` - Voice → text
- `WebSocketManager.swift` - OpenClaw gateway connection
- `TTSService.swift` - Text → voice (3 backends)
- `Message.swift` - Chat message model
